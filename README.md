## Requirements

- iOS 10.0+
- Xcode 11+

## Installation

LVTEKYCsdk is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LVTEKYCsdk'
```
